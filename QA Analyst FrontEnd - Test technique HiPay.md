# QA Analyst - Test technique

# CONTEXTE

Vous êtes QA Analyst dans la squad **Front-End** chez HiPay. Votre équipe développe et maintient les interfaces du site institutionnel et du dashboard marchand. Votre mission est de garantir que les composants UI sont fonctionnels, accessibles, et que le parcours utilisateur est fluide sur tous les supports.

**Support de test** : [HiPay - Fraud Management](https://hipay.com/en/our-solutions/fraud-management/#contact)

## 2. AUTOMATISATION (Partie Pratique)

**Objectif** : Automatisez 1 à 2 scénarios de tests "End-to-End" (E2E) sur la page fournie que vous avez identifiés précédemment.

● **Outils imposés** : Utilisez un outil de développement piloté par le comportement (BDD) et le modèle d'objet de page pour concevoir vos tests (PS: CodeceptJS est le standard HiPay) avec **Gherkin**.

● Créez un rapport d'exécution.

● Enregistrez votre projet dans Gitlab, Github ou tout autre outil de gestion de dépôt.

### Livrables :

Un rapport d'exécution (Allure Report ou équivalent).

● Le code source sur un dépôt (GitHub/GitLab).
